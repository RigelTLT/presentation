CSS Grid Layout
What is CSS Grid Layout?
A grid is an intersecting set of horizontal and vertical lines that form columns and rows.
Examples of using

            .container {
              display: grid;
            }

Splitting into columns


    .container {
      display: grid;
      grid-template-columns: 200px 200px 200px;
    }

Using fixed and flexible dimensions


    .container {
      display: grid;
      grid-template-columns: 200px 1fr 1fr;
    }
repeat() 

            .container {
              display: grid;
              grid-template-columns: 1fr 1fr 1fr;
              grid-template-columns: repeat(3, 1fr);
            }
minmax() 

            .container {
              display: grid;
              grid-template-columns: repeat(3, 1fr);
              grid-auto-rows: minmax(100px, auto);
            }
